Java继承详解举例

Java继承是什么？

首先要来一段来自于网上的解释：

继承是Java面向对象编程技术的一块基石，因为它允许创建分等级层次的类。
继承就是子类继承父类的特征和行为，使得子类对象（实例）具有父类的实例域和方法，或子类从父类继承方法，使得子类具有父类相同的行为。

生活中的继承

兔子和羊属于食草动物类，狮子和豹属于食肉动物类。
食草动物和食肉动物又是属于动物类。
所以继承需要符合的关系是：父类通用，子类具体。
虽然食草动物和食肉动物都是属于动物，但是两者的属性和行为上有差别，所以子类会具有父类的一般特性，但也会具有自身的特性。

如果子类没有super关键字，子类构造器还是会先引用父类，寻找不带参数的构造器并初始化其内容，然后回到子类自己来进行剩下部分的初始化。

继承特性

  1. 子类继承父类所有非private 的属性和方法，虽然可以用Protected来解决，但一般不推荐。Protected（）允许自己访问，同一个包内访问以及子类可以访问（即使不在一个包内）。

  2. super关键字： super是用来实现对父类成员的访问，引用当前对象的父类

  3. this关键字：this则是指向自己的引用

  4. final关键字： final 既可以修饰方法也可以修饰类。 如果修饰类，类的属性便为不可继承，最终类。如果修饰方法，方法不能被子类重写。
